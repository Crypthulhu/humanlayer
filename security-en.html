Security

Security

Enterprise-grade controls protect every decision and every sensitive dataset processed by HumanLayer.

Last updated: February 2026 security@humanlayer.systems

Your data security is our highest priority. HumanLayer Systems Inc. implements enterprise-grade security measures to protect sensitive information processed by our platform.

Secure infrastructure

  • Encryption in transit: All communications are encrypted via TLS 1.3
  • Encryption at rest: Stored data is encrypted with AES-256
  • Hosting: SOC 2 Type II certified cloud infrastructure
  • Isolation: Each client benefits from a logically isolated environment

Authentication and access

  • Multi-factor authentication (MFA) required for all Sentinels
  • Role-based access control (RBAC)
  • Time-limited sessions with automatic logout
  • Full logging of all actions

Sentinel verification

  • Rigorous identity verification before activation
  • Validation of professional qualifications and authorizations
  • Periodic compliance checks
  • Full traceability of each decision

Regulatory compliance

  • GDPR (General Data Protection Regulation)
  • Law 25 (Quebec)
  • SOC 2 Type II (certification in progress)
  • ISO 27001 (planned)

Vulnerability reporting

If you discover a security vulnerability, please contact us immediately at security@humanlayer.systems. We commit to handling any report within 24 hours.

Audits and testing

  • Regular penetration tests by independent third parties
  • Continuous automated vulnerability scanning
  • Secure code reviews
  • Documented and tested incident response plan
